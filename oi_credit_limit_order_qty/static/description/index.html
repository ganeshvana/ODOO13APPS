Credit Limit Based on Value - Ordered Quantity

Steps to Enable Credit Limit Value

Step 1:

Option to enable credit limit is available in Settings-->Users Menu.

Step 2:

Enable credit limit applicable field and value can be mentioned in credit limit field available in partner master(Sales Menu--Orders-->Customers).

Step 3:

choose sales menu-->orders-->create order and make sure products chosen contains ordered quantities as invoicing policy in product master.

Step 4:

Warning is raised on confirmation when order value exceeds the limit defined and the order gets confirmed-->Create Invoice.

Step 5:

On posting the invoice, warning is raised when credit limit value exceeds the limit defined in customer master.
