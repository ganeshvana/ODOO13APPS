Payment Adjustment Amount Against Invoice

This module allows to get actual amount paid against respective invoice. Choose Accounting-->Vendors-->Payments-->Payment Receipt Report. Additional column actual paid amount appears in the report.

Step 1:

Install the module for payment receipt.

Step 2:

Goto Vendor Payments and print payment receipt.

Additional column actual amount paid can be viewed.

This column fetches actual amount adjusted in the payment against invoice.
